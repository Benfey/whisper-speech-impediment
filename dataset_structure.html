Required Dataset Structure

Directory Structure

data/ ├── raw/ # Your original files go here │ ├── recording1.wav # Audio file │ ├── recording1.txt # Transcription file │ ├── recording2.wav # Audio file │ ├── recording2.txt # Transcription file │ └── ... ├── processed/ # Auto-generated by prepare_data.py ├── text/ # Auto-generated by prepare_data.py └── audio_paths/ # Auto-generated by prepare_data.py

Requirements

  • Audio Files:
    • Format: WAV
    • Duration: Less than 30 seconds per file
    • Location: data/raw/
  • Transcription Files:
    • Format: Plain text (.txt)
    • Must have same name as audio file (just .txt instead of .wav)
    • Contains word-for-word transcription
    • Location: data/raw/

Example

Audio file: data/raw/recording1.wav

Content: [Audio of someone saying "Hello world"]

Transcription file: data/raw/recording1.txt

Content: Hello world

Important Notes

  • Only place your files in the data/raw/ directory
  • The other directories (processed/, text/, audio_paths/) will be created automatically
  • Each audio file must have a matching transcription file
  • Transcription must match the audio exactly
